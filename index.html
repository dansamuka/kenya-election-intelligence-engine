Overview

What this is, and what it isn\u2019t

This is

  • Aggregate, source-cited election-poll tracking
  • Transparent about which figures are hosted, which are missing, and how each record was verified
  • A sandbox for stress-testing coalition scenarios \u2014 not a prediction

This is not

  • Not a voter-targeting or persuasion tool
  • Not a certified or official IEBC/KNBS product
  • Not a seat or election forecast \u2014 scenario outputs are sensitivity tests only
